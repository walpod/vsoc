About

Atan is an Interface to Soccer-Server of Robocup's simulation league. It should allow you (and allowed me) to concentrate on the job of controlling your clients without worrying about the following features offerd by Atan. As programming language Java was choosen. If you want to use Atan you also have to use Java.

  • Connetion to Soccer-Server via UDP
  • Parsing of command strings coming from Soccer-Server
  • Generating of command strings that can be understood by Soccer-Server
  • Side convertion. You dont have to worry on what side you are playing at the moment.

Atan is part of the vsoc project that tries to develop Sserver clients controlled by neuronal nets.

Development

If you think atan schould be improved, you are welcome to participate as an atan-developer. Just drop me a note and if you have a sourceforge id I will authorize you to add your changes to the CVS repository.

How to make atan clients running:
  1. Download and install sserver. Downloadpage: go->.
  2. Download atan. Downloadpage: go ->
  3. Extract the zipfile to an arbitrary directory (ATAN_HOME).
  4. Within the ATAN_HOME directory execute 'java -classpath atan.jar atan.sample.Simple1Main' and 'java -classpath atan.jar atan.sample.Simple2Main'.
  5. Start the sserver and soccer-monitor.
